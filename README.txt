The Availability Calendar Rules module allows you to set an availability 
calendar state using Rules. This module aims to fulfill the many requests 
for setting Availability Calendar states in the Availability Calendar issue 
queues.

Requirements

- Availability Calendars
- Rules

Usage

Add a new rule with the new action "Set an Availability Calendar Status".
Choose a calendar to update.
Choose a field to use as start date, or define a set date.
Choose a field to use as end date, or define a set date.
Choose a status to set.
